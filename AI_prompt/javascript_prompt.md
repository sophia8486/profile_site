# 実装したい機能と要件
参考サイトのような「洗練されたテック感」を出すため、以下の機能を実装してください。ライブラリ（jQueryなど）は使わず、**モダンなVanilla JS（ES6+）**で記述してください。

1. **スクロール連動アニメーション (Intersection Observer)**:
   - ユーザーがスクロールして各セクション（About, Skillsなど）が画面内に入ったタイミングで、ふわっと浮き上がる（CSSクラスを付与する）処理を入れてください。
   - 以前出力したCSSに、このためのクラス（例: `.visible` や `.fade-in-up`）がなければ、追加すべきCSSコードも合わせて教えてください。

2. **スムーススクロール**:
   - ナビゲーションのリンクをクリックした際、目的のセクションまで滑らかに移動するようにしてください。

3. **ハンバーガーメニューの開閉（スマホ用）**:
   - スマホ表示時にメニューボタンを押すと、ナビゲーションがスライドイン/アウトする機能を実装してください。

4. **インタラクティブな演出（重要）**:
   - UnityやXR開発を行っている私のポートフォリオらしさを出すため、**「マウスカーソルに追従するさりげないストーカー（円形のエフェクトなど）」**または**「背景に幾何学模様がゆっくり動くキャンバス描画」**など、少し技術力を感じる演出を1つ加えてください。
