# デザインの重要要件
参考サイト（https://room.bokoko33.me/）のような世界観を再現するため、以下のスタイルを徹底的に作り込んでください。

1. **全体像**:
   - `style.css` として記述してください。
   - 参考サイトのような**ミニマルかつ没入感のあるデザイン**にしてください。
   - 基本配色は**ダークテーマ**（背景は黒に近いグレー #1a1a1a など）で、文字は読みやすい白系。
   - アクセントカラーに彩度を抑えた青や紫などを使い、知的な印象に。

2. **具体的な演出**:
   - **Glassmorphism（すりガラス効果）**: コンテンツの背景に薄い半透明のレイヤーを使い、奥行きを出してください。
   - **Fade-in Animation**: ページ読み込み時に、要素がふわっと浮き上がるようなアニメーションを入れてください。
   - **Hover Effects**: ボタンやリンクにカーソルを合わせた際、滑らかに色が変化したり、少し拡大したりする動きをつけてください。
   - **レスポンシブ**: スマホで見た時も余白が美しくなるように調整してください。
